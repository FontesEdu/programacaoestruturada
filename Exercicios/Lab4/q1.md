# Q1

Processo de criação de um ambiente virtual em Python:

1. Abra o terminal ou prompt de comando.
2. Navegue até a pasta ou diretório onde deseja criar o ambiente virtual.
3. Execute o comando: 
    python -m venv (nome_do_ambiente)
4. Depois de criar o ambiente virtual, ative-o:
    - No Windows:
        (nome_do_ambiente)\Scripts\activate
    - No macOS/Linux:
        source (nome_do_ambiente)/bin/activate
5. Com o ambiente virtual ativado, você pode instalar as dependências necessárias usando pip:
    pip install (nome_do_pacote) Ex: Django, Flask, etc.
6. Para desativar o ambiente virtual, basta usar o comando:
    deactivate

Feito tudo isso é só desenvolver o projeto.

Um ambiente virtual é útil para garantir que as dependências
do projeto sejam isoladas e não afetem outros projetos
ou o sistema global. Ele permite que você tenha diferentes versões
de pacotes para diferentes projetos, evitando conflitos e
Facilitando a gestão das dependências.
